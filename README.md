# Smart-Net-Data-Compatibility

[![NuGet](https://img.shields.io/nuget/v/Usa.Smart.Data.Compatibility.svg)](https://www.nuget.org/packages/Usa.Smart.Data.Compatibility)

Compatibility bridge between [Smart.Data.Accessor](https://github.com/usausa/Smart-Net-Data-Accessor) and [Smart.Data.Mapper](https://github.com/usausa/Smart-Net-Data-Mapper).

Provides `AccessorPropertySelector`, which enables `Smart.Data.Mapper` to use the same property name resolution conventions (including `[Name]` attribute and pascal-case matching) as `Smart.Data.Accessor`.
